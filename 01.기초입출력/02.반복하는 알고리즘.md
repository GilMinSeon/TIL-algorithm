## 1) 1부터 n까지 정수의 합 구하기

### 1-1) while문
```python
n = int(input())

sum = 0
i = 1

while i <= n: # i가 n보다 작거나 같은 동안 반복
  sum += i
  i += 1

print(sum)
```

### 1-2) for문
```python
n = int(input())

sum = 0
for i in range(1, n+1):
  sum += i

print(sum)
```
